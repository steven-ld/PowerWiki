# 更新日志

所有重要的项目变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-12-21

### 新增
- ✨ 初始版本发布
- ✨ 支持从 Git 仓库自动拉取 Markdown 文档
- ✨ 自动同步功能（支持定时同步）
- ✨ 目录树结构展示
- ✨ 文章目录（TOC）自动生成
- ✨ 代码语法高亮
- ✨ 搜索功能
- ✨ 响应式设计（完美适配移动端）
- ✨ 模板系统（Header/Footer/Home）
- ✨ 飞书文档风格界面设计
- ✨ 点击标题返回首页功能
- ✨ URL 路由支持，支持直接访问和刷新
- ✨ 移动端菜单和遮罩层

### 技术特性
- 基于 Express.js 的轻量级服务器
- 无需数据库，直接读取 Git 仓库
- 支持多级目录结构
- 自动解析 Markdown 文件
- 支持多种代码语言高亮

### 配置
- 支持通过 config.json 配置
- 支持自定义同步间隔
- 支持自定义模板
